# 🏨 The Wild Oasis

**The Wild Oasis** is an internal hotel management system designed to streamline hotel operations through a clean and intuitive dashboard.

The application enables staff members to manage cabins, reservations, guest information, and check-in/check-out workflows while providing real-time business insights through interactive charts and statistics.

Although this project was built while following **Jonas Schmedtmann's Ultimate React Course**, it involved extensive debugging, problem-solving, and independent learning that significantly strengthened my understanding of modern React development.

---



## 🚀 Features

### 🏡 Cabin Management

* ➕ Create new cabins
* ✏️ Edit cabin details
* 🗑️ Delete cabins
* 🖼️ Upload and manage cabin images
* 💰 Manage pricing and discounts

### 📅 Booking Management

* 📖 View all bookings
* 🔍 Filter and sort reservations
* 📝 Manage booking status
* 📊 Track reservation activity

### 🛎️ Check-In / Check-Out

* ✅ Guest check-in workflow
* 🚪 Guest check-out workflow
* 📍 Booking status tracking

### 👥 Guest Management

* 🧑 Guest profiles
* 📋 Booking history
* 🌍 Nationality tracking

### 📊 Dashboard Analytics

* 📈 Revenue statistics
* 🏨 Occupancy rates
* 💵 Sales overview
* 📅 Booking trends
* 📉 Interactive charts

### 🔐 Authentication & Security

* 🔑 User authentication
* 🛡️ Protected routes
* 👤 User management
* 🔒 Row Level Security (RLS)

### 🎨 User Experience

* 🌙 Dark Mode
* ⚡ Fast UI updates
* ⏳ Loading states
* 🚨 Error handling
* 📱 Responsive design
